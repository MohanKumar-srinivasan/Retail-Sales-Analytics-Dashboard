# Retail Sales Analytics Dashboard

## Project Overview
Analyzed Superstore sales data using Python and Power BI.

## Tools Used
- Python
- Pandas
- NumPy
- Power BI
- Excel

## KPIs
- Total Sales
- Total Profit
- Total Orders
- Profit Margin %

## Dashboard Features
- Sales by Category
- Profit by Category
- Monthly Sales Trend
- Sales by Region
- Top 10 Products
